# 💊 MedReminder

MedReminder is a Flask-based medicine reminder web application designed to help users manage their medication schedules efficiently.  
The application allows users to add, edit, and delete medicine reminders while providing real-time popup alerts and sound notifications.

---

## ✨ Features

- ➕ Add medicine reminders
- ✏️ Edit existing reminders
- ❌ Delete reminders
- ⏰ Real-time digital clock
- 🔔 Popup alert and beep notification system
- 📱 Responsive and modern user interface
- 🧾 Reminder count tracking

---

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Medicine-reminder-app.git
```

### 2️⃣ Navigate to the project directory

```bash
cd Medicine-reminder-app
```

### 3️⃣ Install Flask

```bash
pip install flask
```

### 4️⃣ Run the application

```bash
python app.py
```

### 5️⃣ Open in browser

```bash
http://127.0.0.1:5000
```
